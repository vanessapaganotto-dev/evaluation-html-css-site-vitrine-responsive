# d-angle - Responsive HTML/CSS Website

---

## Description

**FR :**  
Site vitrine responsive développé en HTML et CSS.  
- Grille fluide pour un design adaptable à toutes les tailles d’écran.  
- Menu mobile animé avec ouverture et fermeture accessibles.  
- Formulaire de contact accessible et fonctionnel.  
- Footer adaptatif avec liens de navigation et principes de responsive design.

---

## Fonctionnalités

- Responsive grid system / Système de grille responsive  
- Mobile-friendly menu with ARIA labels / Menu mobile avec labels ARIA  
- Contact form with validation / Formulaire de contact avec validation  
- Clean, semantic HTML5 structure / Structure HTML5 propre et sémantique  
- CSS for layout and animations / CSS pour la mise en page et animations  

---

## Usage

- Ouvrir le fichier `index.html` dans un navigateur moderne.  
- Le site s’adapte automatiquement à la taille de l’écran.  
- Utiliser le menu burger en haut à droite pour naviguer.  
- Remplir le formulaire pour envoyer un message (fonctionnalité à connecter côté serveur si besoin).
